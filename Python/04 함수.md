### 함수
- 코드를 기능으로 묶은 단위 (코드의 기능 단위)  
- D.A.C (Divide & Conquer)  

- 인자 : 여러개 (O)  
- 리턴값 : 1개 또는 없음(종료 or 화면에 출력)

- 변수의 범위  
  > 지역변수 : 함수 내에서 사용 / 함수 내에서 선언  
  > 전역변수 : 함수 밖에서도 사용 / `global` 키워드로 선언 / **권고(X)**(함수는 기능으로 묶은 것이기에)  
  
- 정의  
  ```python
    def 함수이름(인자):
         실행코드
         리턴값
  ```
