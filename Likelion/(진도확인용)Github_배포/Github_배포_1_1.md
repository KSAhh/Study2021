# github에서 Repository 생성
1. git init  
2. git remote add origin repository주소  
3. git add .  
4. git commit -m "설명"  
5. git push origin master  

- 변경된 사항이 있을 경우, "git add ."부터 다시

          git add .  
> . // 모든 것
> add // 업로드
> local 상에 파일을 얹어놓은 상태

      git commit -m
> -m // 설명을 적겠다
> 아직 push되지 않은 상태
